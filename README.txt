Project Name: Crossword Puzzle Game

Description:
This project is a crossword puzzle game implemented in the C language. 
It allows users to play crossword puzzles by filling in the corresponding words. 
The game features a user-friendly interface and custom words dictionary.
It is designed to provide an enjoyable and interactive crossword puzzle experience for players.

You can add your owm words and crossword format by modifying crossword.txt and dictionary.txt
